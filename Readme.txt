Javascript Learning. Seriado #4.                   

Contenido: "SUMAR números aleatorio, comprobar y emitir resultados con Javascript"
------------------

1. Se crearan tres INPUT, dos en modo READONLY, para cargar allí números aleatorios
producidos por la función Math.random()
2. El otro INPUT es para que el usuario coloque el resultado de la suma de los dos númeroas aleatorios
3. Se valiarán las sumas y en un DIV aparte se colocarán los resultados y un mensaje de si la suma fue
correcta o no
4. Así mismo, se colocará color rojo al texto, si la respuesta fue incorrecta o verde si fue correcta
5. Se crearán funciones para no repetir codigo, y las mismas se ejecutarán en la función de INICIO del
la app Javascript.
6. Se contruyeron dos APPs que hacen los mismo pero de diferente manera!
