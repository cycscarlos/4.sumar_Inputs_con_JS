
Javascript Learning!

Seriado #4.                   

Contenido:

 "SUMAR números aleatorios, comprobar operación y emitir resultadoscon Javascript"

Temática:

1. Se crearan tres INPUT, dos en modo READONLY, para cargar allí los números aleatorios
producidos por la función Math.random()
2. El otro INPUT es para colocar el resultado de la suma de los dos númeroas aleatorios
3. Se validarán las sumas y en un DIV aparte se colocarán los resultados y un mensaje de
la operación; es decir, si la suma fue o no correcta
4. Así mismo, se le colocará color rojo al texto (modificación de atributos CSS con Javascript),
si la respuesta fue incorrecta o color verde, si la misma fue correcta
5. Se crearán funciones para no repetir codigo y establecer buenas prácticas de programación; 
las mismas se ejecutarán en la función de INICIO de la app Javascript.
6. Se programaron dos APPs que hacen los mismo pero de diferente manera!
